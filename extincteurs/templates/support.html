FAQ - Questions Fréquentes

Comment ajouter un extincteur dans l'application ?

Pour ajouter un nouvel extincteur, allez dans la section "Gestion des extincteurs" et remplissez le formulaire avec les informations requises.

Comment effectuer une inspection ?

Allez dans la rubrique "Inspecter" et suivez les étapes du formulaire pour valider les différents points d'inspection.

Comment recevoir des notifications d'inspection ?

Les notifications sont activées par défaut. Vous recevrez des rappels avant chaque inspection prévue selon le calendrier défini dans la base de données.
